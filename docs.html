Documentation

Everything you need to install, configure, and master AccessVeil Pro. All 20 modules explained in detail.

🚀 Installation

  1. Download the plugin — Get accessveil-pro.zip from your Envato purchase downloads page.
  2. Upload to WordPress — Go to Plugins → Add New → Upload Plugin. Choose the zip file and click Install Now.
  3. Activate the plugin — Click Activate Plugin. You'll see "AccessVeil Pro" appear in your WordPress admin sidebar.
Pro Tip After activation, the plugin automatically creates default settings. All 20 modules are enabled by default — you can customize which ones to show.

🔑 License Activation

AccessVeil Pro requires a valid license key to unlock all 20 modules. Without activation, the widget will not appear on your site.

  1. Navigate to AccessVeil Pro → License
  2. Enter your Envato purchase code (format: AVPRO-XXXXX-XXXXX-XXXXX-XXXXX)
  3. Click Activate License
Important Your purchase code is available in your Envato downloads page. Each license is valid for one WordPress installation.

⚡ Quick Start

Once your license is active, follow these quick steps:

  1. Go to AccessVeil Pro → Modules
  2. Toggle which modules you want to offer visitors
  3. Ensure "Enable Widget" is checked in Global Settings
  4. Visit your site — the accessibility button appears automatically

🎨 Appearance Settings

Customize the widget look under AccessVeil Pro → Appearance:

  • Widget Position — bottom-right, bottom-left, top-right, top-left
  • Brand Color — hex color picker affects button, toggles, and accents
  • Widget Size — small (44px), medium (54px), large (64px) trigger button
  • Z-Index — control widget stacking order (default: 99999)

✏️ Custom CSS

Add custom styles in the Appearance tab. All widget elements use the .avp- prefix:

/* Make the trigger button larger */
#avp-widget .avp-trigger {
    width: 80px !important;
    height: 80px !important;
}

/* Change panel width */
#avp-widget .avp-panel {
    width: 400px;
}

/* Customize toggle colors */
.avp-toggle[aria-checked="true"] {
    background: #ff6b6b !important;
}

⚡ Hooks & Filters

Extend AccessVeil Pro programmatically using WordPress hooks:

// Modify settings before rendering
add_filter('avp_settings', function($settings) {
    $settings['widget_color'] = '#ff6b6b';
    return $settings;
});

// Add or remove modules
add_filter('avp_modules', function($modules) {
    // Remove a module
    return array_filter($modules, function($m) {
        return $m['id'] !== 'tooltips';
    });
});

// Do something after settings are saved
add_action('avp_settings_saved', function($settings) {
    error_log('AccessVeil Pro settings updated');
});

📎 Shortcode

Manually place the accessibility trigger anywhere using [avp_trigger]:

[avp_trigger text="Accessibility Options" class="my-custom-class"]

This is useful for adding the trigger in custom header menus or specific page locations.

❓ Frequently Asked Questions

Q: Can I use this on multiple sites?
A: One Regular License covers one WordPress installation. Purchase additional licenses for more sites.

Q: Does it affect performance?
A: No. Assets are under 50KB total, loaded only when enabled. Zero impact on Core Web Vitals.

Q: Is it GDPR compliant?
A: Yes. No cookies, no tracking, no external API calls. Preferences stored in browser sessionStorage.

Q: Works with page builders?
A: Yes — Elementor, Divi, Gutenberg, WPBakery, Beaver Builder, and any standard WordPress theme.
