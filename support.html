Frequently Asked Questions

How do I install Clariti?

Download clariti.zip from Envato. Go to Plugins → Add New → Upload Plugin. Choose the file, install, and activate. Then enter your license key under Clariti → License.

Widget not showing?

Check: (1) License is active, (2) Enable Widget is on in settings, (3) Clear any caching plugins, (4) Not hidden on mobile while testing.

Multiple sites?

One Regular License = one WordPress installation. Purchase additional licenses for more sites.

Does it affect performance?

No. Assets are under 50KB total. No external API calls. Zero impact on Core Web Vitals.

GDPR compliant?

Yes. No cookies. No tracking. Preferences stored in browser sessionStorage only.
